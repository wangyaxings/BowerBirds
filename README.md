# BowerBirds简介
希望该软件能够当您像园丁鸟一样收集亮闪闪的东西时提供便利  
批量提取网页界面上的阿里云分享链接和提取码
# 使用方法
![image](https://user-images.githubusercontent.com/38448350/131888090-d7349eab-cff0-4326-a62c-3a8e6b24e160.png)

（1）复制含有阿里云盘的任意网址；  
（2）粘贴到网址栏后，点击OK；  
（3）提取出的所有阿里云盘分享链接和提取码已经复制到剪切板上了  

# 接下来
（1）打包成exe，方便使用  
（2）自动转存  
（3）百度云链接
